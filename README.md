# LearnSphere 
LearnSphere is an interactive online learning platform developed using HTML, CSS, and JavaScript.
This project provides students with an engaging learning experience through modern UI design, interactive course cards, enrollment forms, search functionality, dropdown menus, and popup-based navigation.
---
## 🚀 Features
* Interactive Homepage
* Course Cards with Hover Effects
* Course Details Popup
* Enrollment Form Popup
* Login Popup
* Search Courses Feature
* Responsive Design
* Dropdown Navigation Menu
* About Section Popup
* Professional UI Design
---
